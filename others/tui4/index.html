单选

  1. 周年纪念文献：

    1. 《资本论》 —— 揭示 剩余价值学说，被誉为 “工人阶级圣经”
    2. 《共产党宣言》 —— 标志着 马克思主义的正式诞生

  2. 信念 属于 非理性因素，非理性因素 具有 激活、驱动和控制 作用

    非理性因素 一般有：假设，猜想，信念，意识，情感，冲动，欲望，本能，联想，想象，直觉，灵感

  3. 剩余价值率 的两个计算公式：m’ = m / c = 剩余劳动时间 / 必要劳动时间

    拓展一下基础知识：

    1. 利润率 的计算公式：p’ = p / (c+v)
    2. 在数值上 p = m
    3. 价格 = 生产成本(c+v) + 平均利润 + (垄断利润)
      因此垄断市场下，价值的表现形式会改变（加上垄断利润）但本质还是围绕价值上下波动的
      可以理解为一个函数在纵轴上有的偏移量

  4. 生产集中：生产资料、劳动力和商品的生产日益集中于少数大企业的过程
    资本集中（圈钱）：大吞小，小并大
    生产集中和资本集中是资本家追逐剩余价值和自由竞争的结果
    经济危机 的 本质：生产相对过剩；根本原因 是：生产社会化和生产资料资本主义私人占有的矛盾

    1. 自质：最本质特征 是 高水平的自立自强
    2. 无关：关键 是 经济循环畅通无阻
    3. 内急：战略基点 是 扩大内需
    4. 攻略：战略方向 是 供给侧结构性改革

    三新 之间的关系：

    1. 新发展阶段 是 现实依据
    2. 新发展理念 是 行动指南
    3. 新发展格局 是 战略选择 （抉择、先手棋、举措）

    建设 现代化经济体系 的主要任务：

    1. 坚实基础 是 大力发展 实体经济
    2. 战略支撑 是 实施 创新 驱动发展战略
    3. 重要基础 是 实施 乡村振兴 战略
