肖四整理

Created2021-12-09|Updated2021-12-13|Policy
|Post View:

卷一

单选

  1. 认识世界的目的是为了改造世界

    人类认识世界改造世界的目的是：满足人类自身需要

    实践三种类型：

    1. 物质生产实践（最基本）
    2. 科学文化实践
    3. 社会政治实践

  2. 社会意识的相对独立性：

    1. 不完全同步性
      1. 先进意识：预测、预言、马克思主义
      2. 落后意识：重男轻女（落后的社会意识对社会发展起到阻碍作用）
    2. 能动反作用：社会意识可以反过来作用于社会存在，指导实践改造社会存在

  3. 利润的本质：剩余价值

    利润掩盖了 剩余价值 与 可变资本（劳动力价值）的关系

    干扰项：资本家认为 利润 是 全部垫付资本 的产物（然而实际上是工人的剩余价值创造的，即可变资本创造的）

    利润和剩余价值的来源相同：都是由雇佣工人剩余劳动创造

  4. 社会再生产 的 核心 ：

    1. 价值补偿（对生产投资后要收到利润才会再投资从而再生产）
    2. 实物补偿（生产资料消耗的补充）

  5. 三新一高，必背内容：

    “十四五” 时期经济发展的 主题：推动高质量发展（三新一高的高）

    高质量发展 的 主线：深化供给侧结构性改革

    构建 新发展格局：

    1. 自质：最本质特征 是 高水平的自立自强
    2. 无关：关键 是 经济循环畅通无阻
    3. 内急：战略基点 是 扩大内需
    4. 攻略：战略方向 是 供给侧结构性改革

    三新 之间的关系：

    1. 新发展阶段 是 现实依据
    2. 新发展理念 是 行动指南
    3. 新发展格局 是 战略选择 （抉择、先手棋、举措）

    建设 现代化经济体系 的主要任务：

    1. 坚实基础 是 大力发展 实体 经济
    2. 战略支撑 是 实施创新驱动发展 战略
    3. 重要基础 是 实施 乡村振兴 战略
    4. 制度保障 是 深化经济体制 改革
    5. 空间布局 是 推动 区域 协调发展
    6. 国际竞争力 是 发展 开放 型经济

  6. 新时代党的民族工作，必备内容

    1. 出发点、落脚点：中华民族伟大复兴
    2. 重要任务：建设社会主义现代化国家
    3. 主线：铸牢中华民族共同体意识

  7. 三次分配，必备内容

    1. 一次分配 看 市场（工资）
    2. 二次分配 看 政府（低保、医保）
    3. 三次分配 看 慈善（捐款）

  8. 全党要 牢记 的 根本问题 是：中国共产党是什么、要干什么 （老干妈）

  9. 《辛丑条约》 的签订标志着清政府已经 彻底放弃 了抵抗外国侵略者的念头，甘当“洋人的朝廷”

  10. 第一次论战：

    1. 维新派 vs 守旧派（资产阶级思想 vs 封建主义思想）
    2. 内容：要不要变法

    第二次论战：

    1. 维新派 vs 革命派
    2. 内容：划清了 革命 与 改良 的界限

  11. 资产阶级革命派 的弱点、错误，根源上讲是因为 自身的软弱性和妥协性

  12. 《正处》必背内容

    《关于正确处理人民内部矛盾的问题》明确提出要走一条 有别于苏联 的中国 工业化 道路。

    《关于正确处理人民内部矛盾的问题》中提出敌我矛盾是 对抗性 矛盾，要用 专政 的方法来解决；人民内部矛盾是 非对抗性 矛盾，要用 民主 的方法来解决。

    人民内部矛盾 的 民主 解决方式主要有：

    1. 政治思想领域： 团结-批评-团结 的方针；
    2. 物质利益分配方面： 统筹兼顾、适当安排 的方针；
    3. 科学文化领域： “百花齐放、百家争鸣” 的方针；
    4. 对于共产党和民主党派的矛盾： “长期共存、互相监督” 的方针。

    社会主义国家 政治生活 的 主题 是 关于正确处理人民内部矛盾的问题 ，这一论断的 根本着眼点 在于 调动一切积极因素，团结一切可以团结的力量 ，把全党的注意力转到社会主义建设上来。

  13. 评价人生价值 的 根本尺度 —— 规律 和 历史

    评价人生价值 的 普遍标准 —— 劳动贡献

  14. 全面依法治国 ：

    1. 总目标：建设 中特法治体系 ，建设 法治国家
    2. 重点任务
      1. 建设中特法治体系
        1. 完备的法律规范体系 是 前提
        2. 高效的法治实施体系 是 重点
        3. 完善的党内法规体系 是 本质要求
      2. 建设法治国家
        1. 三依 共同推进（依法治国、依法执政、依法行政），三法一体 建设（法治国家、法治政府、法治社会
        2. 能不能 依法治国 的 关键 是：依法执政 和 依法行政
        3. 法治国家 是法治建设的 目标，法治政府 是 重点，法治社会 是 基础
    3. 指导方针：
      1. 前提 —— 科学立法
      2. 基础 —— 全民守法
      3. 重点 —— 公正司法
      4. 关键 —— 严格执法

  15. 十九届六中全会 的 最重要成果 是 第三次决议 ——《xxx百年xxx决议》

  16. 中美关系：

    1. 奥巴马：不冲突不对抗、相互尊重、合作共赢
    2. 川普：无
    3. 拜登：和平共处、相互尊重、合作共赢

    不冲突不可能了，尽量和平共处，不要打仗吧，美国坏坏

多选

  1. 哲学上的物质，其 时间、空间 既是 有限的、又是 无限的

    区别于：具体物质形态 的 时间、空间 是 有限的 （现在坐在电脑前的我，终将死去）

    物质、运动、时间、空间具有内在统一性：

    1. 物质是运动的载体
    2. 运动是物质的根本属性
    3. 时间、空间是物质运动的存在形式

    不存在离开时间和空间的物质运动

  2. 价值评价 以客体对主体的价值大小为对象，以 善、美 为认识目的

    知识性认识 以客体的属性、规律为对象，以 真 为目的

    虽然具有 主观性、有差异 ，但评价结果仍有 正确、错误 之分（读书有没有用）

    符合真理的价值评价就是正确认识

  3. 价值规律的作用（四选三）

    1. 自发地调节生产资料和劳动力在社会各生产部门之间的分配比例
      1. 供 > 求，撤离
      2. 供 < 求，涌入
    2. 自发地刺激社会生产力的发展（个别劳动时间与社会必要劳动时间关系）
    3. 自发地调节社会收入分配

    价值规律的消极影响（四选三）

    1. 导致社会资源浪费
    2. 导致收入两极分化
    3. 阻碍技术的进步

  4. 金融资本形成的主要途径（两参一连）（四选三）

    1. 金融联系
    2. 资本参与
    3. 人事参与

    金融寡头控制国家（单选）

    1. 经济上：“参与制” （控制其他行业的股份）
    2. 政治上：同政府 “个人联合”
    3. 文化上：建立政治咨询机构

  5. 新经济政策 的 特征 ：以发展 商品经济 为主要特征的新经济政策

    内容：

    1. 用 粮食税制 取代 余粮收集制
    2. 允许 私人自由贸易 ，恢复 商品货币 关系
    3. 允许 私人小工业企业 发展，采取一些 国家资本主义 的形式来发展生产

    标志：列宁正在 探索 一条符合俄国国情的建设社会主义道路

    意义：

    1. 扭转了国家的严重危机，活跃了苏维埃的 城乡经济
    2. 加强了苏维埃的社会主义 经济基础
    3. 改善了工人、农民和其他劳动者的 物质文化生活

    干扰项：

    1. 战时共产主义
    2. 斯大林模式（国防、重工业、第一个五年计划）
    3. 列宁晚年 提出了 建设社会主义的新构想：
      1. 合作社的形式
      2. 大工业（不是重工业）
      3. …… （多选肯定全选）

  6. 党成立以来的四个伟大的成就：

    1. 新民主主义革命 — 根本社会条件
    2. 社会主义革命 — 根本政治前提和制度基础
    3. 改革开放和社会主义现代化建设 — 体制保证和快速发展的物质条件
    4. 新时代中特 — （三个更为）更完善的制度，更坚持的物质基础，更主动的精神力量

    改革开放（站起来到富起来） —— 新时代中特（站富到强起来）

    第一个百年奋斗目标：全面小康

  7. 习近平新时代中特回答的问题是（四选三，十九届六中新说法）：

    1. 什么是中特，怎样发展中特
    2. 什么是社会主义现代化强国，怎样建设社会主义现代化强国
    3. 什么是长期执政的马克思主义政党，怎样建设长期执政的马克思主义政党

    干扰项：

    1. 邓论：什么是社会主义，怎样建设社会主义
    2. 三个代表：建设什么样的党，怎样建设党
    3. 科学发展观：实现什么样的发展，怎样发展

  8. 新发展阶段

    1. 社会主义进程中的一个重要阶段
    2. 从站富到强起来的阶段
    3. 经过几十年的积累、站到了新的起点上的一个阶段（量上）
    4. 仍然是中国特色社会主义 初级阶段（质上）

    干扰选项：所有国家实现社会主义现代化必须经历的阶段（生产力强的国家完全不用，要自己探索）

  9. 中特考生态环保，多选全选

  11. 第二国际主张：放弃暴力革命，主张合法议会斗争，工人阶级通过选举掌握国家政权

    马克思主义中国化：六届六中全会

    2. 个人理想以社会理想为指引

单选

    商品拜物教（四选三）：

    1. 私有制商品经济条件 下劳动产品只有采取 商品 的形式才能进行 交换，人类劳动的等同性只有采取同质的价值形式才能在交换中体现出来
    2. 劳动量只有采取 价值量 这一物的形式才能进行计算和比较
    3. 生产者的劳动关系的社会性质只有采取商品之间，即物与 物之间相互交换的形式 才能间接地表现出来，这就使 人们之间一定的社会关系被物与物的关系所掩盖，具有了拜物教性质

    是 简单商品时期 的产物，此时还没有货币，只能 物与物 进行交换
    干扰项：资本主义经济，货币

  4. 垄断会抑制市场上价格的自由波动

  5. 新经济政策 的 特征 ：以发展 商品经济 为主要特征的新经济政策

    内容：

    1. 用 粮食税制 取代 余粮收集制
    2. 允许 私人自由贸易 ，恢复 商品货币 关系
    3. 允许 私人小工业企业 发展，采取一些 国家资本主义 的形式来发展生产

    标志：列宁正在 探索 一条符合俄国国情的建设社会主义道路

    意义：

    1. 扭转了国家的严重危机，活跃了苏维埃的 城乡经济
    2. 加强了苏维埃的社会主义 经济基础
    3. 改善了工人、农民和其他劳动者的 物质文化生活

    干扰项：

    1. 战时共产主义
    2. 斯大林模式（国防、重工业、第一个五年计划）
    3. 列宁晚年 提出了 建设社会主义的新构想：
      1. 合作社的形式
      2. 大工业（不是重工业）
      3. …… （多选肯定全选）

  6. 国家资本主义 是 私人资本主义经济 向 国营经济 过渡的一种经济形态（不是独立的经济形态）

    独立的经济形态(3)：个体经济、私人资本主义经济、国营经济

    经济成分的性质 取决于 生产资料所有制

    2. 共同富裕是一个 长远目标
    3. 不同人群不仅实现富裕的程度有高有低，时间上也有先有后，不同地域 富裕程度还会 存在一定差异
    4. 这是一个在 动态中先前发展的过程 ，要持续推动，不断取得成效

    可能出干扰选项的说法：

    1. 不是少数人的富裕，也不是整齐划一的平均主义
    2. 不是所有人都同时富裕，也不是所有地区同时达到一个富裕水平
    3. 不可能 一蹴而就，不可能 齐头并进

  12. 中国共产党早期组织活动：（研宣建讨）

    2. 到工人中去进行宣传和组织工作
    3. 进行 关于建党问题的讨论和实际组织工作

    3. 因此，一开始就是一个以马克思列宁主义为基础的党，是一个区别于第二国际旧式社会改良党的新型工人阶级革命政党

单选

  1. 重点著作记忆（最近几年第一题必考）：

    1. “家族合著喷人”：《神圣家族》 —— 马克思和恩格斯第一部合著作品，批判黑格尔派主观唯心主义
    2. “得粒屎”：《德意志意识形态》 —— 第一次系统阐述历史唯物主义
    3. “公共”：《共产党宣言》 —— 马克思主义公开问世
    4. “穷人要资本”：《资本论》 —— 工人阶级圣经，阐述了剩余价值理论
    5. “百度”：《反杜林论》 —— 百科全书
    6. “德法两转变”：《德法年鉴》—— 从唯心向唯物，从革命民主主义向共产主义的转变

  2. 实践 是客观规律性与主观能动性统一的 基础

    自然界和人类社会都具有客观实在性

    社会基本矛盾 决定 社会历史的变化发展（keywords）

  3. 根据资本不同部分在 剩余价值生产中的不同作用 划分为 不变资本 和 可变资本

    根据资本不同部分在 周转中的不同方式 划分为 固定资本 和 流动资本

    根据资本 在循环中执行的不同职能 划分为 货币资本、生产资本、商品资本

  4. 资本主义简单再生产 （原有规模上重复生产）的 实质 是 物质资料再生产 (重新买)与资本主义 生产关系再生产 (雇佣劳动关系)的统一

    简单商品 再生产 的 实质是 劳动过程和价值形成过程的统一

    资本主义生产过程的 两重性：劳动过程 与 价值增殖过程 的统一

  5. 毛泽东思想开始形成的标志：在理论上阐述了 中国的革命道路（农村包围城市，武装夺取政权）

    毛泽东思想趋于成熟的标志 ：在理论上阐述了 新民主主义理论

    毛泽东思想 （是群主）：实事求是（精髓）、群众路线（党的根本路线）、独立自主

    中国革命的首要问题 ：分清敌友

  6. 全面升华改革 的 出发点和落脚点 ：促进社会公平正义、增进人民福祉

    全面升华改革 的 总目标 ：完善和发展中国特色社会主义，推进国家治理体系治理能力现代化

    中国梦 的 本质 ：国家富强、民族振兴、人民幸福 （人民幸福 是 国家富强、民族振兴 的 根本出发点和落脚点）

    为人民谋幸福、为民族谋复兴 是 新发展理念 的 “根” 和 “魂” 、 党的初心和使命 、党领导现代化建设的出发点和落脚点

  7. 新时代党的 政治建设 的 首要任务：全党服从中央

    新时代 党建 的 总要求：（先主统治在三点）

    1. 以长期 执政能力建设，先进性、纯洁性建设 为 主线
    2. 以 政治建设 为 统领 （摆在首位）
    3. 以坚定 理想信念宗旨 为 根基
    4. 以调动全员积极性、主动性、创造性为 着力点

  8. 党 百年奋斗 得出（今年必考的建党100周年）

    1. 历史结论：走自己的路
    2. 根本成就：中特

  9. 辛亥革命之所以失败

    1. 根本原因：在帝国主义时代，在半殖民地半封建的中国，资本主义的建国方案行不通
    2. 主观原因：从根源上来讲，是中国民族资产阶级的 软弱性和妥协性
      1. 没有提出反帝反封的革命纲领
      2. 不能充分发动和依靠人民群众
      3. 不能建立坚强的革命政党

  10. 土地革命 时期建立的苏维埃政权的性质是 工农民主专政

    抗日战争 时期建立的政权性质是 抗日民主政权

    解放战争 时期建立的政权性质是 各个革命阶级的联合专政

    新中国成立初期 建立的政权性质是 人民民主专政

    思想路线 —— 延安整风运动

    第一次提出了 新民主主义革命的科学概念和总路线 的内容是 《中国革命和中国共产党》

  12. 社会主义改造期间，民族资产阶级仍然 具有两面性 （一面拥护党的领导，一面剥削人民）

    对抗性矛盾 是 敌我矛盾，与民族资产阶级之间的矛盾属于 人民内部矛盾，是 非对抗性 的

    前三年完成了民主革命的遗留任务后，中国的基本矛盾就变成了无产阶级和资产阶级的矛盾，因此需要进行社会主义革命，把资本主义经济变成国有经济

  13. 社会道德建设 中具有 基础性作用 的是 个人品德

    3. 不可违抗性（讲 负面，任何身份犯了法都要收到制裁，区别于第一条）
